# Portfólio Pessoal — React

## Sobre o projeto

Este projeto é um **portfólio pessoal desenvolvido em React**, com o objetivo de apresentar meus conhecimentos em desenvolvimento front-end, projetos práticos e informações profissionais de forma clara e organizada.

O foco do projeto é **simplicidade, boa organização de código e design limpo**.

---

## Tecnologias utilizadas

* React (Create React App)
* JavaScript
* HTML5
* CSS3 (responsivo, sem frameworks)
* Git e GitHub

---

## Funcionalidades

* Apresentação pessoal
* Listagem de projetos com descrição e tecnologias
* Seção de habilidades técnicas
* Informações de contato
* Layout responsivo (mobile, tablet e desktop)
